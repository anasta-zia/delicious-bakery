# SweetHomeBakery - Сайт доставки домашней выпечки

Проект для лабораторной работы по созданию SEO-оптимизированных посадочных страниц.

## Описание
Сайт доставки домашней выпечки и сладостей с двумя посадочными страницами:
- Главная страница (`index.html`)
- Страница каталога (`catalog.html`)

## Особенности
1. **SEO-оптимизация:**
   - Уникальные title и description для каждой страницы
   - Семантическая разметка (header, main, section, article, footer)
   - Оптимизированные заголовки H1-H6
   - Атрибуты alt для всех изображений
   - Структурированные данные

2. **Технические особенности:**
   - Адаптивный дизайн
   - Фильтрация товаров в каталоге
   - Модальное окно заказа
   - Плавная прокрутка

## Как запустить локально
1. Клонируйте репозиторий
2. Откройте `index.html` в браузере
